1. What it does

Cheradip AI Coding Agent is a VS Code extension that helps developers build, debug, and ship software faster. It combines chat, agent tools, composer-style multi-file edits, and optional Cheradip AI or your own cloud provider keys.

  • Chat about your codebase with @-context mentions
  • Agent mode that can read/write files (with your permission)
  • Composer for multi-file changes
  • Inline edits, diffs, checkpoints, and background tasks
  • Team-friendly quotas and pay-as-you-go on paid plans

2. Features

  • Providers — Cheradip AI, OpenAI, Anthropic, Google, Groq, Mistral, DeepSeek, Ollama (local)
  • Included keys — when signed in, managed provider keys can be used without pasting secrets into the UI
  • Semantic search — optional embedding index for your workspace
  • Git sync — optional connect / pull / push around agent tasks
  • Billing — Free → Pro → Plus → Business with request and line-edit meters

4. Install

  1. Open VS Code → Extensions
  2. Search for Cheradip or install from the marketplace listing
  3. Open the Cheradip chat panel, sign in, and pick a plan if you need more quota
